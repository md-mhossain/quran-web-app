# quran-web-app
A modern Quran Web Application built with Next.js and Node.js, featuring Surah browsing, Ayah reading with Arabic and translation, search functionality, and customizable reading settings.



# 🌙 Quran Web Application

A modern and responsive Quran web application that allows users to read Surahs, explore Ayahs with Arabic text and English translation, search verses, and customize their reading experience.

## 🎯 Features

- 📖 **Surah List Page**
  - Displays all 114 Surahs
  - Shows both Arabic and English names

- 📜 **Ayah Page**
  - View all verses of a selected Surah
  - Includes Arabic text and English translation

- 🔍 **Search Functionality**
  - Search Ayahs by translation text

- ⚙️ **Settings Panel**
  - Arabic font selection (multiple options)
  - Arabic font size adjustment
  - Translation font size adjustment
  - Settings persist using localStorage

- 📱 **Responsive UI**
  - Works on mobile, tablet, and desktop devices

## ⚡ How to Run Locally

### 1. Clone the repository

git clone https://github.com/md-mhossain/quran-web-app.git


### 2. Install dependencies

npm install


### 3. Run frontend

npm run dev


### 4. Run backend (optional)

npm run dev
